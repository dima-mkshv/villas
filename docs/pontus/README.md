# PONTUS EUXINUS — документация проекта

Проектная документация нового региона интерактивного атласа: **античное Черноморское
побережье России и Таманский полуостров** (греческая колонизация, Боспорское царство,
римский протекторат, Византия/Таматарха).

> **Статус: ТОЛЬКО ДОКУМЕНТАЦИЯ.** Код и данные ещё не пишутся. Это план и методология
> для ревью — прежде чем тронуть движок. Решения ниже — предложения; помечены 🟡 там, где
> нужно подтверждение заказчика.

## Зафиксированные решения (из брифа)

| Вопрос | Решение |
|---|---|
| Где живёт | **Мультирегиональный сайт** — переключатель регионов (Израиль / Причерноморье) в одном движке `villas` |
| Маркеры | **По эпохам** (цвет = эпоха), как в проекте Израиля; + форма = тип объекта |
| Охват | **Максимальный** — все городища, поселения, крепости, некрополи, святилища |
| Фокус | Греки → Боспор → Рим → Византия |

## Принцип разделения данных (важно)

- **Сайт** показывает кураторскую выжимку (карточка: суть, фото, эпохи, доступ, 3–7 ссылок).
- **Библиотека** (`research/`) хранит всё, что «наковыряли»: полные источники, разночтения
  локализаций, спорные атрибуции, сырые заметки, статус проверки. Чтобы следующий агент мог
  **рыться, добавлять и исправлять**, не перегружая сайт. См. [05-RESEARCH-LIBRARY.md](05-RESEARCH-LIBRARY.md).

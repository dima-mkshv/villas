# 06 — Инвентарь объектов (черновик)

Предварительный список «что покрыть» при максимальном охвате — по районам, от Тамани на юг.
**Это не финальные данные**, а инвентарь для библиотеки (по нему в Phase 2 заводим досье и
кураторские записи). Эпохи/доступ/значимость — черновые, уточняются по источникам.

Обозначения: эпохи — `arch`(аика) `cls`(ассика) `hel`(линизм) `rom`(Рим) `byz`(Византия);
🏛 музеефицировано · 🟡 локализация спорная.

## F. 🟡 Потенциальное расширение (вне текущего охвата)

- **Крым / Европейский Боспор** (Пантикапей-Керчь, Нимфей, Тиритака, Мирмекий…) — другая
  столица того же Боспора. В бриф не входит («побережье России + Таманский залив»), но
  напрашивается как регион-сосед. Решение — отдельно. 🟡
- **Восточное Приазовье** (устье Дона — Танаис) — самый северный боспорский центр; тоже
  кандидат на расширение.

## Итог по охвату

Ядро (Тамань + Анапа) даёт ~12–18 значимых объектов + десятки поселений/некрополей хоры для
библиотеки. Юг (Новороссийск–Сочи) добавляет гавани периплов и византийские крепости. Этого
с запасом хватает на «максимальный охват»; на карту выносим по значимости и доступности,
полный список ведём в `research/pontus/_index.md`.

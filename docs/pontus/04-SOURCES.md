# 04 — Источники и как их цитировать

Что берём, кому доверяем в первую очередь и как это попадает в карточку (`bibliography`).
Мастер-список по региону будет жить в библиотеке (`research/pontus/_sources.md`); здесь —
методический каркас.

## 1. Иерархия доверия (для атрибуций и метрик)

1. **Раскопочные публикации и отчёты** (ИА РАН, музеи-заповедники) — первичные данные:
   площади, слои, датировки, находки. Высший приоритет для фактов «что и сколько».
2. **Эпиграфика** (КБН / IOSPE-INBS) — имена, титулы, даты, события (цари, посвящения).
3. **Античные авторы** — локализация и исторический контекст; **критично**: периплы и
   географы расходятся, поэтому для спорных точек храним версии, а не одну «истину».
4. **Обобщающие монографии и энциклопедии** (Гайдукевич, Блаватский, Кругликова, Кузнецов,
   Сапрыкин; академические энциклопедии).
5. **Wikipedia / научпоп** — навигация, перекрёстная проверка, ссылки для читателя; **не**
   первоисточник для спорных утверждений.

## 3. Эпиграфика

- **КБН — «Корпус боспорских надписей»** (М.–Л., 1965; **КБН-альбом** иллюстраций, СПб., 2004,
  [bibliotheca-classica.org/cirb](http://www.bibliotheca-classica.org/cirb)). Базовый корпус
  надписей Боспора.
- **IOSPE** В.В. Латышева (*Inscriptiones antiquae Orae Septentrionalis Ponti Euxini*,
  1885–1901) → **новый онлайн-корпус**: [iospe.kdl.kcl.ac.uk](https://iospe.kdl.kcl.ac.uk/index-ru.html).
  С 2021 публикуется как **INBS** (*Inscriptions of the Northern Black Sea*) — цитировать по INBS.
  Удобно: онлайн, с переводами и привязкой к местам → прямые ссылки в `bibliography`.

## 4. Институты, экспедиции, музеи (современные данные)

- **Институт археологии РАН** — [archaeolog.ru](https://archaeolog.ru/) (Фанагорийская и др.
  экспедиции; рук. Фанагории — В.Д. Кузнецов).
- **Музей-заповедник «Фанагория»** — [phanagoria-museum.ru](https://phanagoria-museum.ru/)
  (раскопки, площади, история исследований, энциклопедия).
- **Анапский археологический музей «Горгиппия»** — экспозиция in situ.
- **Таманский музейный комплекс** (Гермонасса/Тмутаракань).
- Журналы/серии: **«Древности Боспора»**, **«Боспорские исследования»**, **«Боспорский
  феномен»**, материалы **«Боспорских чтений»**, **ВДИ** (Вестник древней истории), КСИА, РА.
- Агрегаторы статей: **КиберЛенинка**, **eLibrary**, Academia.edu.

## 5. Картография / геоданные

- **Wikidata / Pleiades** (античные места — стабильные ID и координаты, удобно для сверки).
- Публикации с планами городищ (для `areaHa`, границ).

## 6. Как это попадает в карточку (бриф п.7 — «больше ссылок»)

Карточка объекта получает **два блока** (помимо обычных `links`):

- **«Ссылки»** (`links`) — кликабельное: музей, ИА РАН, Wikipedia (RU/EN), профильный архео-сайт.
- **«Источники / Литература»** (`bibliography`) — список с типами:
  - `ancient` — «Страбон, География XI.2.10», «Псевдо-Арриан, Перипл, 63»;
  - `epigraphy` — «КБН № …» / «INBS …» (со ссылкой на онлайн-корпус, где есть);
  - `modern` — «Кругликова И.Т. Горгиппия. М., 1975» и т.п.

Так у простого пользователя — кнопки «куда сходить/почитать», а у интересующегося —
полноценный научный след (включая Страбона), не перегружая основной текст.

## 7. Правила цитирования (конвенция)

- Античка — автор + произведение + книга/глава (Страбон XI.2.10), по возможности с переводом.
- Эпиграфика — корпус + номер (КБН 1037; INBS …) + ссылка.
- Современное — автор, название, город, год (страницы — в библиотеке, на сайт по желанию).
- Спорные утверждения на сайте сопровождаем формулировкой «по …» / «предположительно»; разбор —
  в [библиотеке](05-RESEARCH-LIBRARY.md).

# research/pontus — реестр объектов (библиотека исследований)

Рабочая база для региона Причерноморье. Сайт показывает выжимку; здесь — полные досье,
источники, разночтения. Статусы: `todo` · `researched` (собрано) · `published` (вынесено на сайт).
Формат досье — [docs/pontus/05-RESEARCH-LIBRARY.md](../../docs/pontus/05-RESEARCH-LIBRARY.md).
Мастер-навигатор по источникам — **[_sources.md](_sources.md)**. Доборка хоры — **[_hora-doborka.md](_hora-doborka.md)**.

> **Состояние на 2026-06-30:** 35 досье собраны ресёрчем; **все вынесены на карту** (`regions/pontus/sites-data.js`, регион живёт на villae-romanae.com/pontus.html).
> 26 основных + 8 в добор «весь Крым и материк» (Херсонес ЮНЕСКО, Керкинитида, Калос-Лимен, Неаполь Скифский, Харакс, Киммерик, Зенонов Херсонес, Раевское) + святилище **Береговой-4** (Тамань, поднято из доборки).
> Координаты, хронология, площади, КБН/IOSPE, античные источники, споры локализации — внутри досье.

## Заметки по выносу на карту (для sites-data.js)
- **Цвет маркера = главная эпоха** (палитра — [regions/pontus/periods.js](../../regions/pontus/periods.js)).
- **Форма = тип:** круг = город/поселение, квадрат = крепость, ромб = некрополь/курган, треуг. = святилище.
- **★** = музеефиц./маст-си: Пантикапей, Фанагория, Гермонасса, Горгиппия, Танаис, Царский курган.
- **🟡 спорная локализация** (`localizationCertain:false`): Корокондама, Апатур, Бата, Торик, Никопсия —
  приблизит. точка + пометка; **Апатур** (не локализован) — возможно только в таймлайн/библиотеку, не точкой.
- **closed/под водой:** Акра (подводно), Куль-Оба (земл. холм). 
- Крым: уместна пометка геополит. контекста (как «Внимание — Зап. берег» в Израиле) — см. [_sources.md](_sources.md) §6.

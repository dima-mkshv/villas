# Гермонасса / Hermonassa

## Идентификация
Второй по значению город азиатского Боспора (после Фанагории). Античн.: Ἑρμώνασσα.
Соврем.: ст. Тамань (северная часть), Темрюкский р-н. Метрополия — **Митилена (Лесбос)**, эолийцы
(основатель Семандр; по др. традиции — ионийцы под Гермоном). **Непрерывная жизнь 2600 лет.**

## Локализация
**45.2191, 36.7141** — certain (городище под ст. Тамань; Wikidata Q4070683).

## Хронология
archaic → classical → hellenistic → roman → late-antique → byzantine → **Таматарха (хазары) →
Тмутаракань (Русь) → Матрега (генуэзцы) → Кунгала (Крым. ханство) → Тамань (Россия)**.
Founded ≈ −575/−570 (монеты с ~530). Слой 12–14 м, непрерывен до наших дней. Главная эпоха —
для атласа **byzantine/Таматарха** (уникальная непрерывность), но расцвет — classical–hellenistic.

## Метрики
- Охранная зона 35 га; раскоп ~3.6 га; ~200 м берега размыто. sizeClass **city**.
- Сохранность partial (под совр. станицей; слой 12–14 м; берег размывается).
- excavationStatus: active — с 1952 непрерывно: Рыбаков, Зеест, Коровина (1971–88), Финогенова
  (1989–2007), **Ильина (с 2007)**. ГМИИ им. Пушкина.

## Доступ
**museum** — Таманский музейный комплекс / Таманский археол. музей (с 1975), ст. Тамань.
Федеральный ОКН №231640434520006 (с 1960). НЕ ЮНЕСКО.

## Находки
Храм Афродиты; жилые кварталы с мозаиками; клад золота IV в. до н.э.; мраморный саркофаг;
**Тмутараканский камень** (1068 — древнейшая датированная др.-русская надпись, найден 1792).
